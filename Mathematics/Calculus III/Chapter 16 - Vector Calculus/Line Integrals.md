### Conservative Vector Fields and Potential Functions
- In a [[Gradient Vector Fields|Conservative Vector Field]], the line integral is path independent. Only the endpoints matter. 
- To use the [[#^8ceec3|Fundamental Theorem]], we need to be able to tell if **F** is a gradient (conservative) vector field and, if it is, we need to be able to find f(~), the potential function.
- Since the [[Curl]] of a gradient vector field is zero, we can use it to test to see if we can use the Fund. Theorem.
